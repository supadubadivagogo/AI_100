문제 설명
요리기업 대장금의 사장인 당신은 우연히 연구소에서 어떤 문서를 발견했습니다. 처음에는 단순히 칼로리가 적힌 요리법인 줄 알았던 문서가, 실은 스파이들이 남긴 비밀문서임을 알았습니다. 당신은 이 비밀문서, 요리코드를 해독해 스파이들의 계획을 파악해야 합니다.

요리코드는 식기(변수), 재료(상수), 요리 과정(연산)으로 구성된 독특한 프로그래밍 언어입니다. 제공되는 해독 가이드를 참고하여 각 요리코드를 실행하고 올바른 값을 도출하세요.

요리코드 해독 가이드
cookingguide.png

1) 용어 설명

식기(그릇): 냄비, 후라이팬, 접시처럼 재료들의 값을 담는 통입니다. 시작할 때는 모두 0입니다.
재료: 각 재료는 미리 정해진 정수 값(칼로리 값) 을 가지고 있습니다. 값은 칼로리 테이블에서 확인할 수 있습니다.
2) 요리 과정 설명

① 재료 넣기

지시: “식기에 재료를 넣는다.”
동작: 재료가 가진 숫자만큼 그 식기에 값이 더해집니다.
(여러 번 넣으라고 쓰여 있으면 그만큼 여러 번 더합니다. “번”이 생략되면 1번입니다.)
예시: “냄비에 소고기를 넣는다.”
소고기의 값이 4라면, 냄비는 4가 됩니다.
이미 냄비에 10이 있었다면, 14가 됩니다.
② 내용물 옮기기

지시: “A의 내용물을 B로 옮긴다.”
동작: B의 값에 A의 값이 그대로 더해지고, A는 비워져서 0이 됩니다.
예시: 냄비가 12, 후라이팬이 5일 때 “후라이팬의 내용물을 냄비로 옮긴다.”
→ 냄비는 17, 후라이팬은 0.
③ 가열하기(분)

지시: “식기를 N분간 가열한다.”
동작: 그 식기의 값이 N배가 됩니다.
예시: 냄비가 40일 때 “3분 가열” → 120.
④ 가열하기(초)

지시: “식기를 N초간 가열한다.”
동작: 60 ÷ N의 값으로 나눕니다. 소수점은 버립니다.
| 초 | 동작 | 결과 의미 | |---|---|---| | 30초 | 2로 나누기 | 절반이 됩니다 | | 20초 | 3으로 나누기 | 3분의 1이 됩니다 | | 15초 | 4로 나누기 | 4분의 1이 됩니다 |

예시: 후라이팬이 100일 때 “30초 가열” → 100을 2로 나눠 50.
⑤ 결과 보기

지시: “식기를 식탁 위에 올려두었다.”
동작: 그 식기의 현재 값을 보여줍니다.
3) 상황에 따라 달리하기(조건)

지시: “만약 A가 B보다 내용물이 많으면, … 그렇지 않으면, …”
동작: 문장에 적힌 비교대로 맞으면 실행, 아니면 다음 지시에 따라 움직입니다.
비교에 쓰는 말
내용물이 많으면: A가 B보다 큽니다.
내용물이 적으면: A가 B보다 작습니다.
내용물이 같으면: A와 B가 같습니다.
내용물이 다르면: A와 B가 같지 않습니다.
많거나 같으면 / 적거나 같으면도 그대로 해석 하면 됩니다.
예시: “만약 냄비가 후라이팬보다 내용물이 많으면, 파를 2번 넣는다. 그렇지 않으면, 아무 것도 하지 않는다.”
4) 레시피(동작 묶음)

만들기: “○○ 레시피: … … 끝.” 처럼 이름을 붙여 자주 쓰는 동작을 한 덩어리로 적어 둡니다.
불러 쓰기: “○○을 만든다.” 또는 “○○을 X번 만든다.”라고 적으면 그 덩어리를 그대로 실행합니다.
예시: “국물내기 레시피: 물 넣기 → 가열 → 끝.” / “국물내기를 2번 만든다.”
5) 짧은 예시

“후라이팬에 떡을 넣는다.” (떡=100이라면 후라이팬 100)
“후라이팬을 30초간 가열한다.” (2로 나누어 50)
“만약 냄비가 후라이팬보다 내용물이 많으면, 파를 2번 넣는다.” (파=3이면 2번 넣어 +6)
“후라이팬의 내용물을 냄비로 옮긴다.” (냄비의 값은 그만큼 늘고, 후라이팬은 0)
“냄비를 식탁 위에 올려두었다.” (냄비 값 보여주기)
6) 자주 묻는 질문

소수점 처리는 어떻게 하나요? 항상 버립니다. 50.9 → 50.
초 단위로 가열하면 왜 값이 줄어드나요? 요리 코드의 규칙입니다. 짧게 데우면 정해진 만큼 줄어듭니다(표를 참고하세요).
여러 번 넣으라고 명시되어 있지 않으면? 1번 넣는 것입니다.
요리 코드 예시: 떡볶이 만들기
칼로리 테이블
- 물 = 5 Kcal
- 고추장 = 20 Kcal
- 떡 = 100 Kcal
- 파 = 3 Kcal

요리시작:
    냄비에 물을 4번 넣는다.
    냄비에 고추장을 넣는다.
    냄비를 3분간 가열한다.
    후라이팬에 떡을 넣는다.
    후라이팬을 30초간 가열한다.

    만약 냄비가 후라이팬보다 내용물이 많으면:
        후라이팬에 파를 2번 넣는다.
    끝.

    후라이팬의 내용물을 냄비로 옮긴다.
    냄비를 식탁 위에 올려두었다.
    후라이팬을 식탁 위에 올려두었다.
끝.
실행 결과
176
0
해석
냄비에 물을 4번 넣는다. → 물(5) × 4 = 냄비: 20
냄비에 고추장을 넣는다. → 고추장(20) = 냄비: 40
냄비를 3분간 가열한다. → 냄비 × 3 = 냄비: 120
후라이팬에 떡을 넣는다. → 떡(100) = 후라이팬: 100
후라이팬을 30초간 가열한다. → 후라이팬 // 2 = 후라이팬: 50
만약 냄비가 후라이팬보다 내용물이 많으면: → 120 > 50 = 참
후라이팬에 파를 2번 넣는다. → 파(3) × 2 = 후라이팬: 56
후라이팬의 내용물을 냄비로 옮긴다. → 냄비: 176(120+56), 후라이팬: 0
냄비를 식탁 위에 올려두었다. → 출력: 176
후라이팬을 식탁 위에 올려두었다. → 출력: 0
유의사항
모든 식기의 초기값은 0입니다.
정수 나눗셈은 소수점을 버립니다.
재료의 칼로리 값은 문제에서 제공되거나 추론해야 합니다.
